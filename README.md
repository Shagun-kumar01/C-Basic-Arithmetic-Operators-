# C-Programming-Basics
This repository contain my basic C programming programs while learning in first Semester.

## Program Included
1. **Addition.c** - Simple C program to add two integers taken from user input.
How it works:
- Takes first number form user using 'scanf'.  
- Takes second number from user using 'scanf'.  
- Adds both numbers using '+' operator  
- Display the sum using 'printf'.

2. **Subtraction.c** - C program to calculate available hotel rooms using substraction.
   How it works:
- Takes toral rooms in hotel from user
- Takes today's check-ins from user
- substracts check-ins from total rooms using '-' operator
- Display left rooms on screen
